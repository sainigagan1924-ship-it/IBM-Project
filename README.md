Responsive Chat UI

A compact, responsive chat interface built using HTML, CSS, and JavaScript, designed to resemble a modern messaging application.

Overview

This project demonstrates a two-person chat UI with smooth animations, swipe-to-reply interaction, emoji support, and a clean black and grey theme. It is optimized for both mobile and desktop use and requires no external libraries.

Features

Two-person chat with left/right aligned message bubbles

Distinct bubble styles with timestamps (HH:MM)

Smooth message animations

Swipe or drag to reply with reply preview

Fixed header with avatar, username, and menu

Dark/light mode toggle and dynamic wallpaper

Emoji picker with direct input

Mobile-first, fully responsive design

Tech Stack

HTML5, CSS3, Vanilla JavaScript

Project Structure
index.html
style.css
script.js

Usage

Clone the repository and open index.html in any modern browser. No setup or dependencies required.

Support

Works on Chrome, Edge, Firefox, Safari, and modern mobile browsers.
